Ice_Thickness
=============

Calculation of accumulated freezing degree days (AFDD) and an estimate of ice thickness based on the Steven equation.

Input data should be a in a comma-separated format.

The program requires three columns of data:
	Column 1: Date, in the format [YYYYMMDD]
	Column 2: Minimum temperature for the day
	Column 3: Maximum temperature for the day

If availabe data is in mean daily temperature, simply input the mean daily temperature
in both columns 2 and 3 of the input data file.
